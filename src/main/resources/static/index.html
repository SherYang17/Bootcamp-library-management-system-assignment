Hello and welcome!

Please follow the instructions below:

  1. Click Authors Create and create yourself a new Author
  2. Click Authors Read to read the list of Authors currently available
  3. Click Authors Delete to delete any Authors you have created
  4. Click Authors Update to update any Authors you have created
  5. Click Books Create to create a new book and assign it to an Author
  6. Click Books Read to read from the list of Books with Authors
  7. Click Books Delete to delete any Books you have created
  8. Click Books Update to update the title of any Books you have created
